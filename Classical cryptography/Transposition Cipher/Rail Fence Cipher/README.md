Rail Fence Cipher 籬笆密碼
===
## Principle
### 波浪狀
選定一個高度(Rails)，把明文字串依照像波浪一樣的方式排列。然後從第一排由左向往右依順序抄寫下密文。

### 矩形
把要加密的明文分成 N 個一組，然後把每組的第 1 個字連起來形成密文。
