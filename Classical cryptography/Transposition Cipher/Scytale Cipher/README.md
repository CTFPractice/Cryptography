Scytale cipher 密碼棒
===
## About
密碼棒是個可使的傳遞訊息字母順序改變的工具，由一條加工過、且有夾帶訊息的皮革繞在一個木棒所組成。在古希臘，文書記載著斯巴達人用此於軍事上的訊息傳遞。
