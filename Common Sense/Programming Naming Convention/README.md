Programming Naming Convention
===
## [Pascal Case](https://en.wikipedia.org/wiki/PascalCase)
將所有單字用底線隔開
```
this_is_pascal_case
```

## [Camel Case](https://en.wikipedia.org/wiki/Camel_case)
### Lower Camel Case
第一個單字小寫開頭，第二個單字後皆大寫開頭
```
thisIsLowerCamelCase
```

### Upper Camel Case
所有單字皆為大寫開頭
```
ThisIsUpperCamelCase
```
